# 🌐 אתר TalkDrove

אתר רשמי של TalkDrove עם דפי מדיניות החזרה וצור קשר.

## 📁 מבנה הקבצים

```
website/
├── index.html          # דף ראשי
├── contact.html        # דף צור קשר
├── return-policy.html  # מדיניות החזרה
└── README.md          # קובץ זה
```

## 🚀 איך להשתמש

### פתיחת האתר
1. פתחו את קובץ `index.html` בדפדפן
2. או הפעילו שרת מקומי:
   ```bash
   # עם Python
   python -m http.server 8000
   
   # עם Node.js
   npx serve .
   
   # עם PHP
   php -S localhost:8000
   ```

### ניווט באתר
- **דף ראשי**: `index.html` - דף הבית עם מידע על השירותים
- **צור קשר**: `contact.html` - טופס יצירת קשר ופרטי התקשרות
- **מדיניות החזרה**: `return-policy.html` - מדיניות החזרה מפורטת

## 🎨 תכונות האתר

### עיצוב
- עיצוב מודרני ונקי
- תמיכה מלאה בעברית (RTL)
- רספונסיבי למובייל
- אנימציות חלקות
- גרדיאנטים יפים

### דפים זמינים

#### דף ראשי (`index.html`)
- ניווט קבוע בראש הדף
- אזור גיבור עם כפתורי קריאה לפעולה
- כרטיסי תכונות
- מידע חשוב
- פוטר עם קישורים

#### דף צור קשר (`contact.html`)
- טופס יצירת קשר מלא
- פרטי התקשרות
- שעות פעילות
- קישורים לרשתות חברתיות
- עיצוב דו-עמודתי

#### מדיניות החזרה (`return-policy.html`)
- מדיניות החזרה מפורטת
- תנאי החזרה
- תהליך החזרה
- זכויות וחובות
- פרטי קשר לשירות לקוחות
