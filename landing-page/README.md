# דף נחיתה - טוני'ס פיצה

דף נחיתה מקצועי ומודרני לאתר הפיצה של טוני'ס, עם עיצוב רספונסיבי ואנימציות מתקדמות.

## תכונות עיקריות

### 🎨 עיצוב מודרני
- עיצוב נקי ומודרני עם צבעים תואמים למותג
- טיפוגרפיה עברית איכותית עם פונט Heebo
- אנימציות חלקות ומתקדמות
- רספונסיבי מלא לכל המכשירים

### 📱 רספונסיביות
- מותאם למובייל, טאבלט ומחשב
- ניווט מותאם למובייל עם תפריט המבורגר
- תצוגה מיטבית בכל גודל מסך

### ⚡ ביצועים
- טעינה מהירה עם אופטימיזציה לתמונות
- אנימציות מבוססות CSS ו-JavaScript
- קוד נקי ומאורגן

### 🔧 פונקציונליות
- טופס יצירת קשר עם ולידציה
- ניווט חלק בין הסקציות
- אנימציות מספרים בסטטיסטיקות
- מערכת התראות מתקדמת

## מבנה הקבצים

```
landing-page/
├── index.html          # דף HTML ראשי
├── styles.css          # קובץ CSS עם כל העיצובים
├── script.js           # קובץ JavaScript עם הפונקציונליות
└── README.md           # קובץ זה
```

## התקנה ושימוש

### 1. העלאה לשרת
```bash
# העתק את כל הקבצים לשרת
cp -r landing-page/ /path/to/your/server/
```

### 2. התאמה אישית

#### שינוי צבעים
ערוך את קובץ `styles.css`:
```css
:root {
    --primary-color: #FF6B35;    /* צבע ראשי */
    --secondary-color: #F7931E;  /* צבע משני */
    --text-color: #333;          /* צבע טקסט */
}
```

#### שינוי תוכן
ערוך את קובץ `index.html`:
- החלף טקסטים
- עדכן תמונות
- שנה פרטי קשר

#### שינוי פונקציונליות
ערוך את קובץ `script.js`:
- הוסף אנימציות חדשות
- שנה התנהגות טפסים
- התאם אפקטים

### 3. SEO ואופטימיזציה

#### Meta Tags
הדף כולל:
- Title מותאם
- Description מפורט
- Viewport meta tag
- Character encoding

#### תמונות
- תמונות מ-Unsplash (איכות גבוהה)
- Alt tags לכל התמונות
- אופטימיזציה לטעינה מהירה

## ביצועים

### Core Web Vitals
- **LCP (Largest Contentful Paint)**: < 2.5s
- **FID (First Input Delay)**: < 100ms
- **CLS (Cumulative Layout Shift)**: < 0.1

### אופטימיזציות
- תמונות מותאמות גודל
- CSS ו-JS ממוזערים
- טעינה א-סינכרונית של תמונות
- Throttling לאירועי scroll
